# Weighted Averge (for Vulcan journal)

*WeightedAverge*

Example

cin>>3

cin>>6>>10         //   tab[0]=6 tab2[0]=10
  
cin>>4>>8         // tab[1]=4 tab2[1]=8

cin>>1>>3         //  tab[2]=1 tab2[2]=3

WeightedAverage(3,tab,tab2)=4,52




# Field of the rectangle on coordinate system

Example
